# my-first-project

A sandbox repository for learning Git and GitHub workflows.

## Getting Started

Clone the repo and start experimenting:

```bash
git clone https://github.com/vaniskitchen/my-first-project.git
cd my-first-project
```

## Contributing

1. Create a branch: `git checkout -b my-change`
2. Make your change and commit it
3. Push and open a pull request
